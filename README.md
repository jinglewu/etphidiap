# etphidiap
Elan Touchpad Firmware Update (HID Interface) For B50
---
    Get ELAN TouchPad firmware version & Update Firmware.

Compilation
--- 
    make: to build the exectue project "etphidiap".
    $ make
   
   
Run
---
Get Firmware Version :
  ./etphid_updater -g
  
Update Firmware : 
  ./etphid_updater -b {bin_file}
